A button that you can rotate. 

How to rotate your button:
  1. Add  UIButton+Rotation to your project
  2. On any UIButton
  3. Use: 
  ```
    .rotate()
  ```
  Optionally:
  
  4. You can specify a time of animation like this (default animation time is 0.7 sec):
  ```
    .rotate(time: 1.2)
  ```
